# Facilities Domain — Commands

Quick reference of every command the facilities domain exposes.
These commands cover transport (vehicles, routes, vehicle-to-route
and student-to-route assignments), dormitories and rooms, inventory
(items, categories, stores, receive, issue, sell), and suppliers.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

| Command                            | Capability                          | Description                                                                                  | Events                                                                       | Idempotent? | Offline? |
| ---------------------------------- | ----------------------------------- | -------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- | ----------- | -------- |
| `CreateVehicle`                    | `Vehicle.Create`                    | Create a vehicle with an optional primary driver.                                            | `VehicleCreated`, `DriverAssignedToVehicle` (if driver)                      | no          | yes      |
| `UpdateVehicle`                    | `Vehicle.Update`                    | Patch a vehicle's mutable fields.                                                            | `VehicleUpdated`                                                             | no          | yes      |
| `AssignDriverToVehicle`            | `Vehicle.AssignDriver`              | Assign a new primary driver to a vehicle; closes the prior assignment.                        | `DriverAssignedToVehicle`                                                    | no          | yes      |
| `DeactivateVehicle`                | `Vehicle.Deactivate`                | Move a vehicle to `Retired` or `Maintenance`.                                                 | `VehicleDeactivated`                                                         | no          | yes      |
| `DeleteVehicle`                    | `Vehicle.Delete`                    | Soft-delete a vehicle with no historical assignments.                                         | `VehicleDeleted`                                                             | no          | yes      |
| `CreateRoute`                      | `Route.Create`                      | Create a route with ordered stops.                                                           | `RouteCreated`, `StopAddedToRoute` (per stop)                                | no          | yes      |
| `UpdateRoute`                      | `Route.Update`                      | Patch a route's fare, distance, or title.                                                    | `RouteUpdated`                                                               | no          | yes      |
| `AddStopToRoute`                   | `Route.AddStop`                     | Append a stop to a route.                                                                    | `StopAddedToRoute`                                                           | no          | yes      |
| `UpdateStopOnRoute`                | `Route.UpdateStop`                  | Patch a stop's name, time, or fare override.                                                 | `StopUpdatedOnRoute`                                                         | no          | yes      |
| `RemoveStopFromRoute`              | `Route.RemoveStop`                  | Remove a stop from a route.                                                                  | `StopRemovedFromRoute`                                                       | no          | yes      |
| `DeleteRoute`                      | `Route.Delete`                      | Soft-delete a route with no vehicle assignments.                                             | `RouteDeleted`                                                               | no          | yes      |
| `AssignVehicleToRoute`             | `Transport.AssignVehicle`           | Assign an active vehicle to an active route for a year.                                       | `VehicleAssigned`                                                            | no          | yes      |
| `UnassignVehicleFromRoute`         | `Transport.UnassignVehicle`         | Release a vehicle from a route; optionally force-releases memberships.                       | `VehicleUnassigned`                                                          | no          | yes      |
| `AssignStudentToRoute`             | `Transport.AssignStudent`           | Add a student to a vehicle-route with optional pickup and drop stops.                        | `StudentAssignedToRoute`                                                     | no          | yes      |
| `UnassignStudentFromRoute`         | `Transport.UnassignStudent`         | Remove a student from a vehicle-route.                                                        | `StudentUnassignedFromRoute`                                                 | no          | yes      |
| `CreateRoomType`                   | `RoomType.Create`                   | Create a room type.                                                                          | `RoomTypeCreated`                                                            | no          | yes      |
| `UpdateRoomType`                   | `RoomType.Update`                   | Patch a room type.                                                                           | `RoomTypeUpdated`                                                            | no          | yes      |
| `DeleteRoomType`                   | `RoomType.Delete`                   | Soft-delete a room type.                                                                     | `RoomTypeDeleted`                                                            | no          | yes      |
| `CreateDormitory`                  | `Dormitory.Create`                  | Create a dormitory with intake capacity.                                                     | `DormitoryCreated`                                                           | no          | yes      |
| `UpdateDormitory`                  | `Dormitory.Update`                  | Patch a dormitory's name, address, intake, or description.                                   | `DormitoryUpdated`                                                           | no          | yes      |
| `DeleteDormitory`                  | `Dormitory.Delete`                  | Soft-delete a dormitory.                                                                     | `DormitoryDeleted`                                                           | no          | yes      |
| `CreateRoom`                       | `Room.Create`                       | Create a room with bed count and per-bed cost.                                                | `RoomCreated`                                                                | no          | yes      |
| `UpdateRoom`                       | `Room.Update`                       | Patch a room's type, bed count, cost, or description.                                        | `RoomUpdated`                                                                | no          | yes      |
| `DeleteRoom`                       | `Room.Delete`                       | Soft-delete a room.                                                                          | `RoomDeleted`                                                                | no          | yes      |
| `AssignStudentToRoom`              | `Room.AssignStudent`                | Assign a student to a bed.                                                                   | `StudentAssignedToRoom`                                                      | no          | yes      |
| `UnassignStudentFromRoom`          | `Room.UnassignStudent`              | Release a student from a room.                                                               | `StudentUnassignedFromRoom`                                                  | no          | yes      |
| `CreateItemCategory`               | `ItemCategory.Create`               | Create an item category.                                                                     | `ItemCategoryCreated`                                                        | no          | yes      |
| `UpdateItemCategory`               | `ItemCategory.Update`               | Patch an item category.                                                                      | `ItemCategoryUpdated`                                                        | no          | yes      |
| `DeleteItemCategory`               | `ItemCategory.Delete`               | Soft-delete an item category.                                                                | `ItemCategoryDeleted`                                                        | no          | yes      |
| `CreateItem`                       | `Item.Create`                       | Create an item with SKU and category.                                                        | `ItemCreated`                                                                | no          | yes      |
| `UpdateItem`                       | `Item.Update`                       | Patch an item's mutable fields (stock is adjusted separately).                               | `ItemUpdated`                                                                | no          | yes      |
| `DeleteItem`                       | `Item.Delete`                       | Soft-delete an item.                                                                         | `ItemDeleted`                                                                | no          | yes      |
| `CreateItemStore`                  | `ItemStore.Create`                  | Create an item store.                                                                        | `ItemStoreCreated`                                                           | no          | yes      |
| `UpdateItemStore`                  | `ItemStore.Update`                  | Patch an item store.                                                                         | `ItemStoreUpdated`                                                           | no          | yes      |
| `DeleteItemStore`                  | `ItemStore.Delete`                  | Soft-delete an item store.                                                                   | `ItemStoreDeleted`                                                           | no          | yes      |
| `ReceiveItem`                      | `Inventory.Receive`                 | Record an item receive against a supplier; increments stock.                                 | `ItemReceived`                                                               | no          | yes      |
| `UpdateItemReceive`                | `Inventory.UpdateReceive`           | Patch an item receive.                                                                       | `ItemReceiveUpdated`                                                         | no          | yes      |
| `CancelItemReceive`                | `Inventory.CancelReceive`           | Cancel an item receive; reverses stock.                                                      | `ItemReceiveCancelled`                                                      | no          | yes      |
| `IssueItem`                        | `Inventory.Issue`                   | Issue an item to a recipient; decrements stock.                                              | `ItemIssued`                                                                 | no          | yes      |
| `UpdateIssueStatus`                | `Inventory.UpdateIssue`             | Update an issue's status (e.g. mark lost).                                                   | `ItemIssueStatusUpdated`                                                    | no          | yes      |
| `ReturnIssuedItem`                 | `Inventory.ReturnIssued`            | Return part or all of an issued item; restocks.                                              | `IssuedItemReturned`                                                         | no          | yes      |
| `SellItem`                         | `Inventory.Sell`                    | Record a sale of items to a buyer; decrements stock.                                         | `ItemSold`                                                                   | no          | yes      |
| `UpdateItemSell`                   | `Inventory.UpdateSell`              | Patch an item sell.                                                                          | `ItemSellUpdated`                                                            | no          | yes      |
| `CancelItemSell`                   | `Inventory.CancelSell`              | Cancel an item sell; reverses stock.                                                         | `ItemSellCancelled`                                                          | no          | yes      |
| `RefundItemSell`                   | `Inventory.RefundSell`              | Issue a partial refund against an item sell.                                                 | `ItemSellRefunded`                                                           | no          | yes      |
| `CreateSupplier`                   | `Supplier.Create`                   | Create a supplier.                                                                           | `SupplierCreated`                                                            | no          | yes      |
| `UpdateSupplier`                   | `Supplier.Update`                   | Patch a supplier's mutable fields.                                                           | `SupplierUpdated`                                                            | no          | yes      |
| `DeactivateSupplier`               | `Supplier.Deactivate`               | Mark a supplier inactive or blacklisted.                                                     | `SupplierDeactivated`                                                        | no          | yes      |
| `DeleteSupplier`                   | `Supplier.Delete`                   | Soft-delete a supplier with no receive history.                                              | `SupplierDeleted`                                                            | no          | yes      |

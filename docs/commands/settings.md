# Settings Domain — Commands

Quick reference of every command the settings domain exposes. These
commands cover general settings, theme and language selection,
language catalog and phrases, base setup, date format, styles,
background and dashboard layouts, custom links, themes, color
catalog, color themes, behavior record, and setup admin catalog.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

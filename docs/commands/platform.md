# Platform Domain — Commands

Quick reference of every command the platform domain exposes. These
commands cover the multi-school tenancy model, users, OTPs, courses,
custom fields, chart of account, base setup, modules and add-ons,
locale catalogs, front-office content, operational helpers, plugins,
comments, personal access tokens, and video uploads. Many commands
require a system tenant context.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

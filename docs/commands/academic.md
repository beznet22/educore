# Academic Domain — Commands

Quick reference of every command the academic domain exposes. Every
command is validated, capability-checked, and dispatched to a single
aggregate. Commands in this catalog cover the student lifecycle, the
class / section / subject catalog, the academic-year machinery, the
class routine, homework, lesson plans, certificates, ID cards, and
admission inquiries.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

The "Idempotent?" column notes whether a duplicate submission has the
same observable effect as a single submission. Idempotent commands
can be safely retried by the offline queue.

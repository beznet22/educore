# Communication Domain — Commands

Quick reference of every command the communication domain exposes.
These commands cover notices, complaints, notifications, email and
SMS logs, templates, gateway configuration, absent-notification
schedules, chat (one-to-one, groups, invitations, block lists),
bulk send messages, contact-form intake, speech sliders, phone-call
logs, and custom SMS settings.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

| `UnpublishNotice`                | `Notice.Unpublish`               | Hide a notice from new notifications.                                                        | `NoticeUnpublished`                                                                 | no          | yes      |

# Finance Domain — Commands

Quick reference of every command the finance domain exposes. These
commands cover the fees catalog (groups, types, master, assign,
discount, installment), payments, direct fees, carry-forward, login
prevention, banks and bank slips, expense and income (and their
heads), donors, wallet, payroll, inventory and product payments,
chart of account, salary template, hourly rate, installment credit,
and assorted configuration.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

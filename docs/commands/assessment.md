# Assessment Domain — Commands

Quick reference of every command the assessment domain exposes. These
commands cover exam types and exams, exam scheduling, marks
lifecycle, result publication, report card generation, online exams,
seat plans, admit cards, exam attendance, teacher evaluation, and
related configuration.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

| Command                          | Capability                       | Description                                                                                          | Events                                                          | Idempotent? | Offline? |
| -------------------------------- | -------------------------------- | ---------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- | ----------- | -------- |
| `CreateExamType`                 | `ExamType.Create`                | Create an exam type with percentage and average mark.                                                | `ExamTypeCreated`                                               | no          | yes      |
| `UpdateExamType`                 | `ExamType.Update`                | Patch an exam type's mutable fields.                                                                 | `ExamTypeUpdated`                                               | no          | yes      |
| `DeleteExamType`                 | `ExamType.Delete`                | Soft-delete an exam type not referenced by any `Exam`.                                               | `ExamTypeDeleted`                                               | no          | yes      |
| `CreateExam`                     | `Exam.Create`                    | Create a per-subject exam under an exam type.                                                        | `ExamCreated`                                                   | no          | yes      |
| `UpdateExam`                     | `Exam.Update`                    | Patch an exam's marks. Locked once marks have been entered.                                          | `ExamUpdated`                                                   | no          | yes      |
| `DeleteExam`                     | `Exam.Delete`                    | Soft-delete an exam with no marks register.                                                          | `ExamDeleted`                                                   | no          | yes      |
| `ScheduleExam`                   | `Exam.Schedule`                  | Schedule an exam's date/time and per-subject slots.                                                  | `ExamScheduled`                                                 | no          | yes      |
| `UpdateExamSchedule`             | `Exam.Schedule`                  | Update an exam schedule with conflict re-checks.                                                     | `ExamScheduleUpdated`                                           | no          | yes      |
| `CancelExamSchedule`             | `Exam.Schedule`                  | Mark a schedule inactive.                                                                            | `ExamScheduleCancelled`                                         | no          | yes      |
| `InitializeMarksRegister`        | `Marks.Initialize`               | Create one `MarksRegister` per student in the section.                                               | `MarksRegisterCreated`                                          | no          | yes      |
| `EnterMarks`                     | `Marks.Enter`                    | Upsert a single (student, subject) mark row.                                                         | `MarksEntered`                                                  | yes         | yes      |
| `SubmitMarks`                    | `Marks.Submit`                   | Lock a marks register and compute grades.                                                            | `MarksSubmitted`                                                | no          | yes      |
| `PublishResult`                  | `Result.Publish`                 | Materialize result rows and merit positions for a section.                                           | `ResultPublished`, `ResultStoreCreated`                         | no          | yes      |
| `RepublishResult`                | `Result.Publish`                 | Re-apply result computation and supersede the prior publication.                                     | `ResultRepublished`                                             | no          | yes      |
| `GenerateReportCard`             | `ReportCard.Generate`            | Build a report card payload for a single student.                                                    | `ReportCardGenerated`                                           | no          | yes      |

| `PublishOnlineExam`              | `OnlineExam.Publish`             | Move an online exam from pending to published.                                                       | `OnlineExamPublished`                                           | no          | yes      |
| `StartOnlineExam`                | `OnlineExam.Start`               | Open an online exam for students in its time window.                                                 | `OnlineExamStarted`                                             | no          | yes      |
| `SubmitOnlineExamAnswer`         | `OnlineExam.Answer`              | Record a student's answer to one question.                                                           | `OnlineExamAnswered`                                            | yes         | yes      |
| `EvaluateOnlineExam`             | `OnlineExam.Evaluate`            | Compute per-student marks; optionally close the exam.                                                | `OnlineExamEvaluated`, `OnlineExamMarkCreated`, `OnlineExamClosed` (optional) | no | yes |
| `GenerateSeatPlan`               | `SeatPlan.Generate`              | Allocate students to rooms for a scheduled exam.                                                     | `SeatPlanGenerated`                                             | no          | yes      |
| `GenerateAdmitCard`              | `AdmitCard.Generate`             | Materialize an admit card for a student.                                                             | `AdmitCardGenerated`                                            | no          | yes      |
| `SetExamSignature`               | `ExamSignature.Set`              | Create or update the exam signature asset.                                                           | `ExamSignatureCreated` / `ExamSignatureUpdated`                 | yes         | yes      |
| `ConfigureCustomResultSettings`  | `Result.Configure`               | Upsert per-exam-type result layout settings.                                                         | `CustomResultSettingUpdated`                                    | yes         | yes      |
| `MarkTeacherEvaluation`          | `TeacherEvaluation.Mark`         | Submit a student rating for a teacher.                                                               | `TeacherEvaluationCompleted`                                    | yes         | yes      |
| `AddTeacherRemark`               | `TeacherRemark.Add`              | Add a class teacher remark for a student.                                                            | `TeacherRemarkAdded`                                            | no          | yes      |
| `UpdateTeacherRemark`            | `TeacherRemark.Update`           | Patch a teacher remark.                                                                              | `TeacherRemarkUpdated`                                          | no          | yes      |
| `ApproveTeacherEvaluation`       | `TeacherEvaluation.Approve`      | Approve a teacher evaluation.                                                                        | `TeacherEvaluationApproved`                                     | no          | yes      |
| `RejectTeacherEvaluation`        | `TeacherEvaluation.Approve`      | Reject a teacher evaluation.                                                                         | `TeacherEvaluationRejected`                                     | no          | yes      |
| `CreateMarksGrade`               | `MarksGrade.Create`              | Add a band to the school's grade scale.                                                              | `MarksGradeCreated`                                             | no          | yes      |
| `UpdateMarksGrade`               | `MarksGrade.Update`              | Patch a grade band.                                                                                  | `MarksGradeUpdated`                                             | no          | yes      |
| `DeleteMarksGrade`               | `MarksGrade.Delete`              | Remove a grade band.                                                                                 | `MarksGradeDeleted`                                             | no          | yes      |
| `MarkExamAttendance`             | `ExamAttendance.Mark`            | Mark present / absent for the exam-day roll.                                                         | `ExamAttendanceMarked`                                          | yes         | yes      |
| `UpdateExamAttendance`           | `ExamAttendance.Mark`            | Update a previously marked exam roll.                                                                | `ExamAttendanceUpdated`                                         | no          | yes      |
| `CreateExamSetting`              | `ExamSetting.Create`             | Configure per-exam-term publish date and window.                                                     | `ExamSettingCreated`                                            | no          | yes      |
| `UpdateExamSetting`              | `ExamSetting.Update`             | Patch an exam setting.                                                                               | `ExamSettingUpdated`                                            | no          | yes      |
| `DeleteExamSetting`              | `ExamSetting.Delete`             | Remove an exam setting.                                                                              | `ExamSettingDeleted`                                            | no          | yes      |
| `ConfigureAdmitCardSettings`     | `AdmitCard.Configure`            | Upsert the school's admit card layout settings.                                                      | `AdmitCardSettingUpdated`                                       | yes         | yes      |
| `ConfigureSeatPlanSettings`      | `SeatPlan.Configure`             | Upsert the school's seat plan layout settings.                                                       | `SeatPlanSettingUpdated`                                        | yes         | yes      |
| `ConfigureTeacherEvaluation`     | `TeacherEvaluation.Configure`    | Toggle and configure the teacher evaluation feature for the school.                                  | `TeacherEvaluationConfigured`                                   | yes         | yes      |
| `PublishExamRoutine`             | `ExamRoutine.Publish`            | Publish an exam routine to the public site.                                                          | `FrontExamRoutinePublished`                                     | no          | yes      |

| `UpdateExamRoutinePage`          | `ExamRoutinePage.Update`         | Patch the public exam routine page.                                                                  | `ExamRoutinePageUpdated`                                        | no          | yes      |
| `UpdateFrontendExamResult`       | `FrontendExamResult.Update`      | Patch the public exam result page.                                                                   | `FrontendExamResultUpdated`                                     | no          | yes      |

| `AddOnlineExamQuestion`          | `OnlineExamQuestion.Add`         | Add a question to an online exam.                                                                    | `OnlineExamQuestionAdded`                                       | no          | yes      |

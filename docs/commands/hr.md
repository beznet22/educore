# HR Domain — Commands

Quick reference of every command the HR domain exposes. These commands
cover the staff lifecycle, departments and designations, class-teacher
assignment, leave, staff attendance (including imports), payroll,
salary templates and hourly rates, staff registration fields, bulk
staff imports, and subject-teacher assignment.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

# CMS Domain — Commands

Quick reference of every command the CMS domain exposes. These
commands cover the public-facing content model: pages, news,
comments, notice boards, testimonials, sliders, home-page settings,
content and content share lists, course / contact / about / news
pages, and related catalogs.

The "Events" column lists the events the command emits; consult the
per-domain spec for payload structure.

| Command                            | Capability                       | Description                                                                                  | Events                                          | Idempotent? | Offline? |
| ---------------------------------- | -------------------------------- | -------------------------------------------------------------------------------------------- | ----------------------------------------------- | ----------- | -------- |
| `CreatePage`                       | `Page.Create`                    | Create a `Page` in `Draft` status.                                                            | `PageCreated`                                   | no          | yes      |
| `UpdatePage`                       | `Page.Update`                    | Patch a page's mutable fields.                                                               | `PageUpdated`                                   | no          | yes      |
| `PublishPage`                      | `Page.Publish`                   | Move a page to `Published` status.                                                           | `PagePublished`                                 | no          | yes      |
| `ArchivePage`                      | `Page.Archive`                   | Move a published page back to `Draft`.                                                       | `PageArchived`                                  | no          | yes      |
| `DeletePage`                       | `Page.Delete`                    | Soft-delete a non-default page.                                                              | `PageDeleted`                                   | no          | yes      |
| `CreateNews`                       | `News.Create`                    | Create a news article.                                                                       | `NewsCreated`                                   | no          | yes      |
| `UpdateNews`                       | `News.Update`                    | Patch a news article.                                                                        | `NewsUpdated`                                   | no          | yes      |
| `PublishNews`                      | `News.Publish`                   | Publish a news article.                                                                      | `NewsPublished`                                 | no          | yes      |
| `UnpublishNews`                    | `News.Unpublish`                 | Unpublish a news article.                                                                    | `NewsUnpublished`                               | no          | yes      |
| `DeleteNews`                       | `News.Delete`                    | Soft-delete a news article.                                                                  | `NewsDeleted`                                   | no          | yes      |

| `DeleteNewsComment`                | `NewsComment.Delete`             | Soft-delete a news comment.                                                                  | `NewsCommentDeleted`                            | no          | yes      |
| `CreateNoticeBoard`                | `NoticeBoard.Create`             | Create a notice board entry.                                                                | `NoticeBoardCreated`                            | no          | yes      |

| `PublishNoticeBoard`               | `NoticeBoard.Publish`            | Publish a notice board entry.                                                                | `NoticeBoardPublished`                          | no          | yes      |

| `DeleteTestimonial`                | `Testimonial.Delete`             | Soft-delete a testimonial.                                                                   | `TestimonialDeleted`                            | no          | yes      |

| `UpdateHomeSlider`                 | `HomeSlider.Update`              | Patch a home slider.                                                                         | `HomeSliderUpdated`                             | no          | yes      |

| `ConfigureHomePage`                | `HomePageSetting.Configure`      | Create or update the school's home page setting.                                             | `HomePageSettingCreated` / `HomePageSettingUpdated` | yes       | yes      |
| `CreateContent`                    | `Content.Create`                 | Upload a content file or YouTube link.                                                       | `ContentCreated`                                | no          | no       |
| `UpdateContent`                    | `Content.Update`                 | Patch a content entry.                                                                       | `ContentUpdated`                                | no          | no       |

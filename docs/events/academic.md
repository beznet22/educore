# Academic Domain — Events

Quick reference of every event the academic domain emits. Events are
immutable, append-only records used for cross-domain integration,
audit, and event sourcing. Every event carries a typed
`EventEnvelope` and is durably persisted to the aggregate's event
log. Most events are replicated to consumers over the event bus
according to the `Subscribers` column.

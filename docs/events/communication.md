# Communication Domain — Events

Quick reference of every event the communication domain emits. Events
are immutable, append-only records. Every event carries a typed
`EventEnvelope` and is durably persisted to the aggregate's event
log. The notification adapter is the canonical fan-out point for
delivery to channels.

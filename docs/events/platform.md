# Platform Domain — Events

Quick reference of every event the platform domain emits. Events
are immutable, append-only records. Every event carries a typed
`EventEnvelope` and is durably persisted to the aggregate's event
log. School lifecycle events seed RBAC, settings, and academic;
user lifecycle events drive RBAC and operations audit.

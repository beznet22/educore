# Operations Domain — Events

Quick reference of every event the operations domain emits. Events
are immutable, append-only records. Every event carries a typed
`EventEnvelope` and is durably persisted to the aggregate's event
log. Job and system-version events are observed by the platform and
settings domains for cache invalidation and version display.

| Event                              | Aggregate           | Subscribers                                                                                  | Description                                                                              | Durable? | Replicated? | Replayable? |
| ---------------------------------- | ------------------- | -------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------- | -------- | ----------- | ----------- |

| `BackupDeleted`                    | `Backup`            | —                                                                                            | A backup record was deleted.                                                             | yes      | yes         | yes         |
| `BackupRestored`                   | `Backup`            | `platform`                                                                                   | A backup was restored.                                                                   | yes      | yes         | yes         |
| `BackupMarkedActive`               | `Backup`            | —                                                                                            | A backup was marked as the school's active backup.                                       | yes      | yes         | yes         |
| `BackupMarkedInactive`             | `Backup`            | —                                                                                            | A backup's active flag was cleared.                                                      | yes      | yes         | yes         |
| `JobScheduled`                     | `Job`               | job runner                                                                                   | A job was scheduled.                                                                     | yes      | yes         | yes         |
| `JobCancelled`                     | `Job`               | job runner                                                                                   | A job was cancelled.                                                                     | yes      | yes         | yes         |
| `JobReserved`                      | `Job`               | job runner                                                                                   | A job was reserved by a worker.                                                          | yes      | yes         | yes         |
